# Spinner for Twister 
Spinner is a automated Spinner for the game of Twister.  This app lets all people be in the mat.

# Running on Windows
## msys-x64
```
go build -v *.go
```
```
go run -v *.go
```

# Building .apk for Android Studio
## msys-x64 
```
fyne package -os android -appID com.example.spinner -icon assets/twister-spinner.png
```
## Installing .apk to Android Studio
Drag spinner.apk to emulator icon

# Building Windows 
```
fyne package -os windows -icon assets/twister-spinner.png

# Building Linux 
```
fyne package -os linux -icon assets/twister-spinner.png
